 مسیر یادگیری بساز (Learning Path)
از [[DAG]] یک مسیر اصلی استخراج کن:
مسیر پیشنهادی:
1. Workflow as State Machine
2. State Persistence
3. Transactions
4. Task Queue Problem
5. Local Transfer Queue
6. Sharding
7. Routing
8. Matching / Worker
9. Visibility
10. Multi-cluster
این مسیر را یک «سری نوت» کن که هر کدام برای مطالعه‌ی روزانه مناسب باشد.
